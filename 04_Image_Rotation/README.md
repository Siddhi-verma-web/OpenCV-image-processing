# 🔄 Image Rotation using OpenCV

## 📌 Overview

This project demonstrates how to rotate an image using OpenCV in Python.

Image rotation is a common image transformation technique used to change the orientation of an image. In this implementation, the input image is rotated in three different directions:

- 90° Clockwise
- 180°
- 90° Counterclockwise

---

## ✨ Features

- Read an input image using OpenCV
- Rotate the image 90° clockwise
- Rotate the image 180°
- Rotate the image 90° counterclockwise
- Display all rotated images
- Save all rotated images automatically

---

## 🛠️ Technologies Used

- Python
- OpenCV
