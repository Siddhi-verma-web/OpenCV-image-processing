# ✂️ Image Cropping using OpenCV

## 📖 Overview

This project demonstrates how to crop a specific region from an image using OpenCV in Python.

Image cropping is one of the most common image processing techniques. It allows you to extract a selected portion of an image for further analysis or processing.

---

## ✨ Features

- Read an image using OpenCV
- Crop a selected region from the image
- Display the cropped image
- Save the cropped image
- Print the cropped image dimensions

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy

---

## 📂 Project Files

- `crop_image.py` - Python implementation
- `input.png` - Original input image
- `output_crop.png` - Cropped output image

---

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install opencv-python numpy
```

2. Run the program:

```bash
python crop_image.py
```

---

## 📸 Output

The program crops the selected region from the original image, displays it in a new window, and saves it as **output_crop.png**.
