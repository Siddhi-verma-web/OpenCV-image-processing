# RGB to Grayscale Conversion

## 📖 Overview
This project demonstrates how to convert a color image into a grayscale image using Python and OpenCV.

Grayscale conversion is one of the most fundamental preprocessing techniques in image processing and computer vision.

---

## ✨ Features
- Read an RGB image
- Convert the image to grayscale
- Display the grayscale image
- Save the grayscale image

---

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

---

## 📂 Files
- `rgb_to_gray.py`
- `input.png`
- `output_gray.png`

---

## 🎯 Learning Outcome
Through this implementation, I learned how OpenCV converts a color image into grayscale and why grayscale images are commonly used before applying advanced image processing techniques.
